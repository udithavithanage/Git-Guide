5. Push and Set Upstream üöÄ

Push to the remote branch and set upstream:

git push -u origin feature-branch

Purpose of `-u`: - `-u` (or `--set-upstream`) links your local branch with the remote branch. - After setting upstream, future pulls and pushes can be done with just `git pull` and `git push`.

6. Pull Latest Changes ‚¨áÔ∏è

To get the latest changes from the remote repository:

Important: `-u` is **not used** with `git pull` because upstream tracking is already set with `git push -u`.

Best Practice: After using `git push -u`, you can simply run:

git pull

7. Merge Branches üîÑ

Merge changes from another branch:
